Novice to Ninja - Chapter 8 Form Focus

The form input below triggers the following event listeners and displays an indication message:

  • Focus - Activated when cursor placed within input
  • Blur - Activated when cursor moved out of input without a change occuring
  • Change - Activiated when cursor moved out of input after a change has occured within the input
