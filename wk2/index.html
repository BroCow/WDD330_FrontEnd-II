JS Ninja Chapter 2 - Programming Basics

JavaScript Grammer

C-style - similar to C languages

ASI automatically adds semi-colons, but it is best practice to add them at the end of each statement.

Blocks - Do not need to end with semi-colons {}

Objects - Anything that isn't a primitive data type

Variables

const = can't change value, but not immutable

Underlying data in object can change (mutating object)
const name = {value: 'Chris'}; name.value = 'Carter';

Try to use const to declare most variables - cuts down on bugs

var = don't use it (outdated)

let = replaced var as of ES6 (block-scoped)

Global scope variables - minimize the use to reduce chance of errors

Backslash - can be used to escape characters like '

Properties - accessed by using the . notation (such as .length) OR brackets ['']

Methods - accessed by .notation followed by (), perform an action (like toUpperCase())

Template Literals - AWESOME! Using backticks (`) allows for insertion of variables into strings using ${}; ES6 often uses backticks to create ALL strings!!!

Number.isInteger - used to check if number is/isn't and integer (boolean)

0x = hexidecimal / 0o = octal / 0b = binary

Number() is preferred way to change string to number: Number('23');

String() is preferred way to change number to string: String(23);

Always use HARD EQUALITY to avoid potential coercion issues. If number is submitted in form, convert to number using Number('5') === 5.

JS Ninja Chapter 3 - Arrays, Logic, and Loops

Arrays

const myArray = [] - Creates an array literal

Delete - only removes the VALUE of the item, not the index itself

Destructuring - const [x,y] = [1, 2] - variables exist outside of array

Array Properties and Methods

.length - shows how many indexes; array[array.length -1] shows last item

Array Length can be adjusted manually - array.length = 5; if there were more items in the array, they would be removed!

join() Method - used to turn array into a string with comma separated values, or takes different separator as an argument ('&')

slice() Method - like a copy and paste of the sliced items into another array (non-destructive)

splice() Method - replaces item in array with another item, or inserts new item at specified location; also a good option for removing item from array completely by specifying location, "1", and no replacement splice(3,1) (destructive)

sort() method - Alphabetic by default, but sorts numbers by 1st digit (10 before 9)

indexOf() method - searches for value in array, returns index number. This would be good for a "search & delete" function!

Multidimentional Arrays

Array of arrays - const coordinates = [[1,3], [5,2]];

To access values - coordinates [1][0] - where first value is which array, second value is index

spread() method can be used to "flatten" Multidimentional arrays

Maps

Good for key-value pair lookup (also called hash tables or dictionaries

.get method returns value of key: map.get(4) returns value of key "4"

Logic

Ternary Operator - shorthand for if-else statement: condition ? //cod to run : //code for else
