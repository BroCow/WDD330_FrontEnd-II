The Browser Object Model

The Browser Object Model (or BOM for short) is a collection of properties and methods that contain information about the browser and computer screen.

The following code provides a quick way of assigning the variable global to the global object:

window.history.length property shows how many pages have been visited before arriving at the current page

window.history.go(-1); // goes back 1 page

The Document Object

Cookies

Small files saved on users computer

A restriction of cookies is that they can only be read by a web page from the same domain that set them

limited to storing up to 4KB of data, although 20 cookies are allowed per domain

Cookies can be used for personalizing a user’s browsing experience, storing user preferences, keeping track of user choices (such as a shopping cart), authentication and tracking users.

Creating Cookies

document.cookie = 'name=Superman';
-- "name=Superman"

More cookies can be added - just extends the string

Changing Cookie Values

Just reassign it using the same name with different value:
document.cookie = 'name=Batman'
-- "name=Batman"
